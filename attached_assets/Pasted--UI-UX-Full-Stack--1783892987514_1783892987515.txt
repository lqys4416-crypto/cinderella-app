أنت فريق تطوير برمجيات محترف يضم محلل أنظمة، مصمم UI/UX، ومطور Full Stack. أريد منك بناء نظام احترافي كامل من الصفر باسم **Cinderella Team** لإدارة فريق التسويق والطلبات والعمولات، مع واجهة عربية حديثة ومتوافقة مع الهواتف والكمبيوتر.

## الهدف

إنشاء نظام احترافي يساعدني على إدارة فريق من المسوقات، بحيث يكون لكل مسوقة حساب خاص بها، وتقوم بإدخال طلبات الزبائن، بينما أستطيع أنا كمدير متابعة الطلبات، تغيير حالتها، وحساب الأرباح والعمولات تلقائيًا.

---

# التقنيات المطلوبة

* Frontend: React + Next.js + TypeScript
* Backend: Node.js + Express
* Database: PostgreSQL
* ORM: Prisma
* Authentication: JWT
* تصميم حديث باستخدام Tailwind CSS
* Responsive 100% لجميع الشاشات
* دعم اللغة العربية RTL
* تصميم فاخر باللون الأسود والذهبي
* أيقونات حديثة
* حماية قوية للبيانات
* كود منظم وقابل للتطوير
