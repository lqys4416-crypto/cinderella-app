# نظام إدارة الأصول (Asset Manager) - الدليل التشغيلي

لقد تم تحويل المشروع بالكامل إلى نظام احترافي لإدارة الأصول. إليك تفاصيل ما تم إنجازه وكيفية البدء.

## التغييرات الرئيسية
1. **الهوية البصرية:** تم استبدال اللون الأسود والذهبي بهوية زرقاء احترافية تناسب أنظمة الشركات.
2. **التطبيق الجديد:** تم تغيير اسم التطبيق إلى **Asset Manager** ومعرف الحزمة إلى `com.asset.manager`.
3. **قاعدة البيانات:** تمت إضافة جداول جديدة لإدارة الأصول (الاسم، التصنيف، الرقم التسلسلي، الحالة، الموقع).
4. **السيرفر:** تم تحديث الـ API لدعم عمليات إضافة وعرض وتعديل الأصول.

## كيفية تشغيل المشروع

### 1. تشغيل السيرفر (Back-end)
انتقل إلى مجلد السيرفر وقم بتشغيله:
```bash
cd artifacts/api-server
pnpm install
pnpm dev
```
*سيعمل السيرفر على المنفذ 3000.*

### 2. تشغيل تطبيق الجوال (Front-end)
انتقل إلى مجلد التطبيق:
```bash
cd artifacts/cinderella-mobile
pnpm install
npx expo start
```
*يمكنك استخدام تطبيق Expo Go على هاتفك لمشاهدة التعديلات فوراً.*

## كيفية استخراج ملف الـ APK النهائي
للحصول على ملف APK جاهز للتثبيت، اتبع الخطوات التالية:

1. **تسجيل الدخول إلى EAS:**
   ```bash
   eas login
   ```
2. **بناء ملف APK للتجربة:**
   ```bash
   eas build --platform android --profile preview
   ```
   *بعد انتهاء البناء، سيعطيك النظام رابطاً لتحميل ملف الـ APK.*

## الميزات المتوفرة الآن
- **شاشة دخول آمنة:** (استخدم `admin` و `admin123` للدخول).
- **لوحة تحكم (Dashboard):** تعرض إحصائيات الأصول (المتاحة، قيد الاستخدام، الصيانة).
- **قائمة الأصول:** عرض جميع الأصول المسجلة مع إمكانية البحث (تم إعداد الهيكل البرمجي).
- **إضافة أصل جديد:** شاشة لإدخال بيانات الأصول الجديدة.
